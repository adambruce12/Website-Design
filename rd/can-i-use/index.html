Can I Use

Can I Use is a site that allows a user to search and see what browsers support specific features. The site provides information such as what browsers support the technology, what percentage of users will be covered based on browser support, and what level of supports the a browser offers (none, partial, or full support. The site also provides information such as notes, known issues, and resources relating to the technology.

Uses of Resource Information

  1. Find out what browsers support a feature the developer would like to implement.
  2. Find additional resources related to specific features.
  3. See the percentage of users that will be using browsers that support a feature.
  4. Keep up to date with new features and technologies related to web development.

Properties

CSS3 Selectors

CSS3 Selectors are advanced element selectors that can be utilized in CSS. These include child(), :target, :empty, and many more selectors. Currently 99.58% of all globabl users use browswers that support CSS3 selectors.

CSS Counters

CSS Counters refers to a method of controlling number values in generated content. Counters utilize the counter-reset and counter-increment properties. CSS Counters are widely supported with a globabl percentage of 99.58% of users using web browsers that support the feature.

Summary

Can I Use is a resource that proves itself valuable time and time again. The information provided lets developers know whether utilizing a new feature is a good idea at all, whether it should be rolled out with additional work arounds for some broswers, or whether a feature is fully-supported and good to utilize. The site is also a great place to find information regarding features and new developments relating to web development as a whole.
