CSS Flex

Flexbox Froggy

Flexbox Froggy is an online game that teaches the concepts of flex via helping frogs get to their lilypads. To get the frogs to their lilypads successfully the player must use different flex properties and values to move the frogs.

CSS-Tricks: A Complete Guide to Flexbox

The CSS-Tricks Complete Guide to Flexbox is a ten-part guide that covers multiple topics including the background of flexbox, flexbox properties, flexbox tricks, browser support, basics and terminology, and additional resources for reference. The author provides a lot of visual representations as well as multiple examples of code.

Free Code Camp: Flexbox-The Ultimate CSS Flex Cheatsheet

Free Code Camp's Flexbox - The Ultimate CSS Flex Cheatsheet is a comprehensive flexbox cheatsheet that covers terminology and properties with the aid of animated gifs to demonstrate what is happening. The article also links to four games (one being Flexbox Froggy) that learners can play to help master flexbox

Summary

There are no shortage of great resources for learning CSS Flex, but the ones listed above are three that I have found personally helpful to learn the concepts and also utilize as quick references. Flexbox Froggy adds an element of fun and both the Free Code Camp and CSS-Tricks articles provide great illustrations and make great reference guides.
