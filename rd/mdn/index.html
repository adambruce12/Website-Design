MDN HTML and CSS Reference

MDN HTML

The MDN HTML reference page describes all elements and attributes relating to HTML. This includes references to elements, attributes, global attributes, link types, content categories, and date and time formats.

MDN CSS

The MDN CSS reference page is an alphabetical index of all the standard CSS properties, pseudo-classes, pseudo-elements, data types, functional notations, and at-rules. It also lists selectors (organized by type) and key CSS concepts.

Summary

MDN's HTML and CSS reference pages offer a helpful guide to both languages that's easy to use based on best practices. MDN
